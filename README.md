# python-shopping-cart-system
A simple Python shopping cart program using dictionaries to add items, manage cart items, display cart contents, and calculate total amount.
# Python Shopping Cart System

A beginner-friendly Python mini project that simulates a shopping cart system using dictionaries.

## Features
- Add items with price
- Add items to cart
- Display cart items
- Calculate total amount
- Menu-driven program

## Technologies Used
- Python 3

## How It Works
The program allows users to:
1. Add shop items and prices
2. Add items into cart with quantity
3. View cart contents
4. Calculate total bill
5. Exit the program

## Concepts Used
- Dictionaries
- Loops
- Conditional statements
- User input handling

## Run the Program

```bash
python shopping_cart.py
